# Lit Accordions

Accordions toggle example

## Usage

```bash
git clone git@github.com:brysonbw/lit-accordions.git
```

```bash
cd lit-accordions
```

```bash
rm -rf .git
```

### Install Dependencies

```bash
npm install
```

## Developing

```bash
npm run dev
```

## Run Tests

### Unit

```bash
npm run test:unit
```

## Building

```bash
npm run build
```
